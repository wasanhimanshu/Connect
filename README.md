# Connect
<p> Connect is a real time one to one chat application made with firebase.</p>

<p>Features</p>
<ul>
  <li>Firebase RealtimeDatabase</li>
  <li>FirebaseAuth</li>
  <li>FirebaseStorage</li>
  </ul>
